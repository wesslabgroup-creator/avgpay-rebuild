# SEO Audit Findings - Iteration 1

## Critical SEO Issues

### 1. Duplicate Title Tags
**Every single page has:**
```
<title>AvgPay - Know Your Worth in 60 Seconds</title>
```

**Should be:**
- Home: "AvgPay - Know Your Worth in 60 Seconds"
- PM Guide: "Product Manager Compensation Guide 2026 | AvgPay"
- Programmatic: "Software Engineer Salary at Google in San Francisco | AvgPay"
- Pricing: "Pricing - Free Salary Insights or PRO | AvgPay"

### 2. Missing Meta Descriptions
**Current:** None visible
**Need:** Unique descriptions for every page

### 3. No Schema Markup
**Missing:**
- JobPosting schema for salary pages
- Article schema for guides
- FAQPage schema for methodology
- BreadcrumbList schema

### 4. H1 Issues
- Multiple H1s on some pages
- H1s don't match title tags

### 5. URL Structure Issues
- Spaces encoded as %20 in titles
- Inconsistent casing

### 6. Missing Technical Elements
- No sitemap.xml
- No robots.txt
- No canonical tags
- No Open Graph images

## Content Gaps
- No FAQ section (high SEO value)
- Thin content on some programmatic pages
- No internal linking strategy
- Missing LSI keywords

## Competitive Opportunities
- Levels.fyi has poor mobile UX
- Glassdoor has paywall friction
- Indeed lacks specificity
- We can win on speed + accuracy

## Immediate Fixes Needed
1. Dynamic title generation
2. Meta description component
3. Schema markup utilities
4. Sitemap generation
5. Robots.txt
6. Fix encoded URLs in display
